## Installation

> [!NOTE]
> Visual Studio Code app is needed.

1. Clone repository on your computer.
2. Run commands

```bash
npm i
npm run server
```

3. Provide public port in VSCode

![VSC public port](./screenshots/2023_12_23_20_43_43_587_mylN.png)
